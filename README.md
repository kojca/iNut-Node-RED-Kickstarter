## Developers

Các bước cài đặt

1. Clone code về:

        git clone https://github.com/ngohuynhngockhanh/iNut-Node-RED-Kickstarter
        cd iNut-Node-RED-Kickstarter

2. Cài đặt

        npm install

3. Build chương trình.

        npm run build

4. Chạy chương trình

        npm start
   hoặc

        node red.js
